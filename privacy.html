PRIVACY POLICY

LAST UPDATED: JUNE 27, 2026

01 — OVERVIEW

This Privacy Policy describes how dn_zeK Stream Notif ("the App") handles information. The App is a personal automation tool used solely by its creator (dn_zeK) to post and delete go-live notifications on connected social media accounts when a Twitch or Kick stream begins or ends.

02 — DATA COLLECTED

The App collects no personal data from any users. It does not operate a website with user sign-in, does not log visitor information, and does not use cookies or analytics tracking of any kind.

The only data the App processes is:

  • Stream status events from Twitch and Kick (online / offline signals)
  • The current game title being streamed (for inclusion in post text)
  • Temporary access tokens stored locally for API authentication

None of this data is shared with third parties or stored beyond the active stream session.

03 — THIRD-PARTY API ACCESS

The App uses the following APIs exclusively to manage content on accounts owned by dn_zeK:

  • TikTok Content Posting API — to post and delete go-live notifications
  • Bluesky AT Protocol — to post and delete go-live notifications
  • YouTube Data API v3 — to post and delete community posts
  • Facebook Graph API — to post and delete Page posts
  • Instagram Content Publishing API — to post feed content

Each API integration operates only on the operator's own accounts. No third-party user data is accessed, stored, or processed.

04 — DATA RETENTION

No user data is retained. API tokens are stored locally on the operator's own machine and are never transmitted to any server other than the respective platform's official API endpoints.

05 — CHILDREN'S PRIVACY

This App is not directed at children and does not knowingly collect information from anyone under the age of 13.

06 — CHANGES TO THIS POLICY

This policy may be updated periodically. The "Last Updated" date at the top of this page reflects the most recent revision.

07 — CONTACT

For privacy-related questions, contact via twitch.tv/dn_zek or @dn-zek.bsky.social.
